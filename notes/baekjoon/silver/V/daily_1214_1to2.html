    문제 6108 — The Perfect Cow

    문제 링크

    백준 문제 보기

    코드 (IDE 스크린샷)

    해설

    The Perfect Cow (BOJ 6108)는
    N×N 격자에 있는 소들의 아름다움 점수에서, 각 행의 중앙값(메디안)을 뽑고, 그 N개의 중앙값 중 다시 중앙값을 구하는 문제다.

    • N은 홀수이므로, 중앙값은 항상 N/2번째(0-index 기준)로 명확하다.
    • 각 행을 정렬해서 중앙값을 구하고, 그 결과 리스트를 다시 정렬해서 중앙값을 출력하면 끝.

    전략 개요

    • 1) 행 단위 중앙값 추출
      • 각 행의 N개 점수를 임시 리스트에 담는다.
      • 리스트를 오름차순 정렬한 뒤, N/2번째 값을 “그 행의 중앙값”으로 저장한다.
    • 2) 중앙값들의 중앙값 계산
      • 행 중앙값이 담긴 리스트(N개)를 정렬한다.
      • 그 리스트의 N/2번째 값을 출력한다.

    코드가 하는 일

    • 각 행을 정렬해서 중앙값을 뽑고 list2에 저장 
      for (int i = 0; i < N; i++) {
          StringTokenizer st = new StringTokenizer(br.readLine());
      
          for (int j = 0; j < N; j++) {
              space[i][j] = Integer.parseInt(st.nextToken());
              list1.add(space[i][j]);
          }
      
          Collections.sort(list1);
          list2.add(list1.get(N / 2));
          list1.clear();
      }
      • list1은 “현재 행”의 값들을 담는 임시 버퍼.
      • 정렬 후 list1.get(N/2)가 그 행의 중앙값.
      • 중앙값은 list2에 누적하고, 다음 행을 위해 list1을 비운다.
    • 행 중앙값 리스트(list2)를 정렬한 뒤 중앙값 출력 
      Collections.sort(list2);
      System.out.println(list2.get(list2.size() / 2));
      • list2에는 총 N개의 행 중앙값이 들어있다.
      • 정렬 후 N/2번째가 “중앙값들의 중앙값” → 정답.

    시간 · 공간 복잡도

    • 시간 복잡도
      • 각 행 정렬: N번 × O(N log N) → O(N^2 log N)
      • 중앙값 리스트 정렬: O(N log N) (위에 비해 매우 작음)
    • 공간 복잡도
      • 입력 저장 N×N + 리스트들 → 대략 O(N^2)

    문제 6123 — O Those Fads

    문제 링크

    백준 문제 보기

    코드 (IDE 스크린샷)

    해설

    O Those Fads (BOJ 6123)는
    초기 매력도 L에서 시작해서, 참여한 소 한 마리당 매력도가 K씩 증가할 때 최종적으로 몇 마리까지 참여하게 되는지를 구하는 문제다.

    • 소 i는 저항치 R_i가 있고, 현재 매력도 A가 R_i 이상이면 참여한다.
    • 참여한 소 수를 cnt라 하면 현재 매력도는 A = L + cnt * K로 계속 증가한다.
    • 매력도는 한 번 올라가면 절대 내려가지 않으므로, 저항치가 낮은 소부터 순서대로 확인하는 게 핵심이다.

    전략 개요

    • 1) 저항치 정렬
      • R_i를 오름차순 정렬한다.
      • 가장 쉽게 참여할 수 있는 소부터 참여시키는 것이 최대로 참여시키는 방법이다.
    • 2) 그리디로 참여 가능 여부 체크
      • 현재 참여자 수를 cnt라고 할 때, 현재 매력도는 L + cnt * K.
      • 정렬된 배열을 앞에서부터 보면서 R_i <= L + cnt * K면 참여시켜 cnt++.
      • 어떤 소가 조건을 만족 못하면(저항치가 너무 크면), 뒤에 있는 소들은 더 크거나 같으므로 즉시 종료한다.

    코드가 하는 일

    • 저항치 정렬 
      Arrays.sort(cows);
      • 저항치가 낮은 소부터 참여 여부를 확인하기 위해 오름차순 정렬.
    • 현재 매력도 계산 + 참여 처리 
      int cnt = 0;
      for (int i = 0; i < N; i++) {
          long A = L + (long) cnt * K;
      
          if (cows[i] <= A) cnt++;
          else break;
      }
      • cnt명 참여했으면 현재 매력도는 A = L + cnt*K.
      • 현재 소의 저항치가 A 이하면 참여 → cnt++.
      • 한 번 실패하면 뒤는 더 큰 저항치이므로 더 이상 참여 불가 → break.
      • long을 쓰는 이유: L + cnt*K가 int 범위를 넘을 수 있어서 안전하게 계산.

    시간 · 공간 복잡도

    • 시간 복잡도
      • 정렬: O(N log N)
      • 한 번 순회: O(N)
      • 총합: O(N log N)
    • 공간 복잡도
      • 저항치 배열: O(N)
