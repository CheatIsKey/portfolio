    CSS 기본 개념

    작성일 2025-10-18 · 분류: CSS

    개요

    회원 등록 HTML 페이지는 구조만 있고 스타일이 없어 구식처럼 보인다. 여기에 CSS를 적용하면 우리가 평소 보는 깔끔한 폼과 레이아웃을 손쉽게 만들 수 있다. CSS는 HTML의 표현(색, 여백, 배치, 반응형)을 담당한다.

    CSS 적용 방법: 3가지

    1) 인라인 스타일 (inline style)

    태그 안의 style 속성에 직접 선언한다.

    <label style="color: red;">이름</label>

    2) 내부 스타일 (internal stylesheet)

    <head> 안의 <style> 블록에 선언한다.

    <head>
  <style>
    label { color: red; }   /* 내부 스타일 */
  </style>
</head>
<body>
  <label>아이디</label>
</body>

    3) 외부 스타일 (external stylesheet)

    별도의 style.css 파일을 링크한다.

    <head>
  <link rel="stylesheet" href="style.css">
</head>

    style.css 파일에는 내부 스타일과 동일한 문법을 사용한다(예: label { color: red; }).

    장단점 요약

    방식장점단점
    인라인 스타일
    • 아주 빠르게 한 요소만 수정 가능
    • CSS 파일 없이도 즉시 테스트
    • 재사용/유지보수 어려움(혼잡)
    • 우선순위가 강해 충돌 유발
    내부 스타일
    • 한 문서 안에서 일관 관리 용이
    • 구현/디버깅이 간단
    • 다른 페이지와 스타일 공유 어려움
    • 문서가 길어지면 가독성 저하
    외부 스타일
    • 여러 페이지에서 재사용 가능
    • 캐싱으로 성능 향상
    • 파일 의존성(링크 깨지면 적용 실패)
    • 초기 로드에 네트워크 의존

    우선순위(특이성) 간단 정리

    • 인라인 > 내부/외부 (단, !important와 구체적 선택자, 선언 순서의 영향도 받음)
    • 선택자 특이성: id > class/속성/가상클래스 > 태그/가상요소
    • 동일 특이성일 때는 나중에 선언된 규칙이 이김

    선택자: id와 class

    <div id="wrapper" class="wrapper">...</div>
    • #wrapper (id): 문서에서 중복 불가, 특정 하나를 정확히 지정
    • .wrapper (class): 중복 허용, 여러 요소에 공통 스타일

    박스 모델(Box Model)

    모든 요소는 박스로 취급된다. 안쪽부터 content → padding → border → margin 순서다.

    • content: width, height로 크기 지정
    • padding: 콘텐츠와 테두리 사이의 내부 여백(배경색 적용됨)
    • border: 테두리 두께/선/색
    • margin: 박스 바깥쪽 간격(요소 간 거리)

    레이아웃: display와 position

    display

    • block: 줄바꿈을 발생, 가로폭(가능한 영역)을 채움
    • inline: 줄 안에 흐르며 내용 너비만 차지(폭/높이 직접 지정 어려움)
    • inline-block: 줄 안에 흐르되 폭/높이 지정 가능
    • none: 렌더 트리에서 제거(보이지 않음, 자리도 차지 X)

    position

    • static: 기본 흐름(좌표 속성 사용 불가)
    • relative: 자기 원래 위치를 기준으로 이동(공간은 유지)
    • absolute: 가장 가까운 position이 static이 아닌 조상 기준
    • fixed: 뷰포트 기준으로 고정(스크롤해도 고정 표시)

    Flexbox 핵심

    부모-자식 관계로 동작한다. 부모에 display: flex;를 선언하면 자식들이 유연하게 줄 바꿈/정렬된다.

    .form {
  display: flex;
  flex-direction: column;   /* row | column | row-reverse | column-reverse */
  gap: 12px;
}

    주요 속성

    • flex-direction: 주 축(main axis) 방향(row가 기본, column은 세로)
    • justify-content: 주 축 정렬(앞/가운데/끝, space-between/around/evenly)
    • align-items: 교차 축(cross axis) 정렬(상단/가운데/하단/늘이기)
    • align-content: 여러 줄로 래핑된 행들의 교차 축 정렬(복수 행에서만 의미)

    권장 사용법: 저작권 표기를 그림 바로 아래 캡션(figcaption)에 링크와 함께 명시하고, 문서 끝의 “참고/출처” 섹션에도 동일 링크를 한 번 더 적어 두면 가장 안전하다.
