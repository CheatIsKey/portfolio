    CSS 실습: 회원가입 폼 스타일링

    작성일 2025-10-19 · 분류: CSS

    준비

    HTML에는 id와 class 선택자를 지정했고, <head>에서 외부 스타일시트 style.css를 연결했다.

    <link rel="stylesheet" href="style.css">

    전역: 변수와 박스 모델

    :root에 색상 변수를 선언하고, 문서 전체 박스 계산을 border-box로 설정한다.

    :root{
  --primaryColor:green;
  --errorColor:red;
  --backgroundColor:rgb(67, 144, 233);
}
html{ box-sizing: border-box; }
    • content-box: padding, border 제외한 콘텐츠만 폭/높이에 반영.
    • border-box: padding, border 포함해 계산(이번 실습에서 사용).

    레이아웃: 화면 중앙 정렬

    body를 플렉스 컨테이너로 만들어 자식(래퍼)을 가로·세로 중앙에 배치한다. min-height:100vh로 뷰포트에 맞춘다.

    body{
  display:flex;
  align-items:center;     /* 수직 중앙 */
  justify-content:center; /* 수평 중앙 */
  min-height:100vh;
  background-color:white;
}

    카드와 타이틀

    #wrapper에 그림자(box-shadow)를 주어 카드 형태를 만든다.

    #wrapper{
  width:420px;
  box-shadow:0 4px 14px rgba(0,0,0,.25);
}
h2{ text-align:center; margin-top:20px; }

    입력 그룹과 상태

    각 필드는 .inputGroup으로 묶고, 라벨/인풋은 한 줄씩 보이도록 display:block을 사용한다.

    .inputGroup{ padding:8px 12px; }
.inputGroup label{ display:block; padding:12px 0; }
.inputGroup input{
  display:block; width:93%;
  padding:12px; border:2px solid #aaa9a9; font-size:16px;
}
.inputGroup input:focus{ outline:none; border-color:#777; }

    valid / invalid 상태

    JS에서 .inputGroup에 .valid 또는 .invalid 클래스를 붙여 테두리 색을 바꾼다. 헬퍼 텍스트는 올바를 때 숨기고, 틀릴 때만 표시한다.

    .inputGroup.valid  input{ border-color:var(--primaryColor); }
.inputGroup.invalid input{ border-color:var(--errorColor); }

.inputGroup .helperText{ display:block; padding-top:8px; }
.inputGroup.valid  .helperText{ visibility:hidden; }
.inputGroup.invalid .helperText{ color:var(--errorColor); }

    버튼

    버튼은 래퍼 안에서 중앙 배치하고, 테마 색 변수를 사용한다.

    #wrapper button{
  display:block; width:50%; height:50px;
  margin:auto; margin-bottom:25px;
  font-size:18px; cursor:pointer;
  background-color:var(--backgroundColor);
  border:2px solid var(--backgroundColor);
  color:white;
}

    완성 스크린샷

    정리

    • :root 변수로 색 테마를 한 곳에서 관리.
    • border-box로 폭 계산 단순화.
    • 플렉스로 화면 중앙 정렬.
    • .valid/.invalid 상태 클래스로 검증 UI 표현.
