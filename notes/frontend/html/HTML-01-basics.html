    HTML 핵심 개념

    작성일 2025-10-18 · 분류: HTML

    개요

    웹페이지는 보통 HTML(구조) + CSS(스타일) + JavaScript(동작)의 조합으로 구성된다. 데이터베이스 연동이 없는 정적 환경이라면 이 세 가지로도 충분히 실습과 학습이 가능하다.

    • HTML: 문서의 뼈대(제목, 문단, 목록, 링크, 이미지 등)와 의미(시맨틱)를 표현한다.
    • CSS: 색상, 여백, 배치, 반응형 등 시각적 표현을 담당한다.
    • JavaScript: 상호작용(폼 검증, 동적 콘텐츠, 이벤트 처리 등)을 구현한다.

    개발 준비물

    VS Code를 사용한다. 추천 확장:

    • HTML CSS Support
    • HTML/CSS/JavaScript Snippets
    • Live Server (저장 시 자동 새로고침로 빠른 피드백)

    HTML 문서의 골격

    현대 표준은 HTML5이며, 문서는 다음 구조를 기본으로 한다.

    <!doctype html>
<html lang="ko">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>문서 제목</title>
  <link rel="stylesheet" href="style.css" />
</head>
<body>
  <h1>안녕하세요!</h1>
  <p>여기에 본문이 들어갑니다.</p>

  <script src="javascript.js"></script>
</body>
</html>
    • <!doctype html>: 현재 문서가 HTML5 규격임을 명시하는 선언(약속).
    • <html lang="ko">: 문서 루트 요소. lang은 주요 언어(한국어) 표시로 접근성/검색에 도움.
    • <head>: 문서 자체의 정보(메타데이터, 스타일, 제목 등).
    • <body>: 화면에 표시될 실제 콘텐츠.

    head의 핵심 메타

    문자 인코딩: UTF-8

    컴퓨터가 사람의 문자를 올바르게 해석하도록 인코딩을 지정해야 한다. 한국어를 비롯한 대부분의 언어를 안전하게 표현하기 위해 UTF-8을 사용한다.

    <meta charset="utf-8">

    반응형 기본값: viewport

    모바일 화면에서도 제대로 보이도록 viewport를 설정한다. 화면 너비를 기기 너비에 맞추고 초기 확대 배율을 1로 둔다.

    <meta name="viewport" content="width=device-width, initial-scale=1">

    문서 제목과 정적 자원

    <title>은 브라우저 탭 제목으로 쓰인다. 정적 파일은 보통 link와 script로 연결한다.

    <title>Frontend 연습</title>
<link rel="stylesheet" href="style.css">
<script src="javascript.js" defer></script>

    팁: defer는 HTML 파싱이 끝난 뒤 스크립트를 실행해 초기 렌더링을 막지 않는다.

    body: 화면에 보이는 콘텐츠

    <body>에는 실제로 보일 텍스트, 이미지, 링크, 폼 등을 배치한다. HTML 요소는 대체로 <tag attr="값">내용</tag> 형태이며, 일부는 닫는 태그가 없는 빈 요소(self-closing)다.

    <h1>제목 1</h1>
<p>문단입니다.</p>
<img src="logo.png" alt="사이트 로고">
<a href="https://example.com">외부 링크</a>

    시맨틱 태그

    HTML5는 의미가 담긴 태그를 제공한다. 이는 접근성, SEO, 유지보수에 유리하다.

    • <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>

    파일 구성 예시

    가장 기본적인 연습 구조는 다음과 같다.

    index.html
style.css
javascript.js

    이 구조만으로도 정적 페이지를 만들고, CSS로 스타일링하며, JS로 간단한 상호작용을 붙일 수 있다.

    정리

    • HTML은 구조, CSS는 스타일, JS는 동작을 담당한다.
    • <!doctype html>, lang, charset, viewport는 HTML5의 기본 세팅이다.
    • 시맨틱 태그를 쓰면 가독성과 접근성이 좋아진다.
