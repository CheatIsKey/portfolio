    작성일 2025-10-22 · 분류: JavaScript

    배열 요소 조작

    배열은 splice(), pop(), delete 등을 통해 요소를 수정·삭제할 수 있다.

    const arr = ["a", "b", "c", "d"];

// 1) splice(index, count) — 삭제
arr.splice(1, 1);  // 1번 인덱스("b") 삭제 → ["a", "c", "d"]

// 2) splice(index, 0, 값) — 추가
arr.splice(1, 0, "x"); // ["a", "x", "c", "d"]

// 3) pop() — 마지막 요소 제거
arr.pop(); // ["a", "x", "c"]

// 4) delete arr[index] — 값만 삭제, 빈칸(undefined) 남음
delete arr[1]; // ["a", empty, "c"]
console.log(arr[1]); // undefined

    주의: delete는 배열의 인덱스 공간을 유지한 채 값을 지워서 희소 배열을 만든다. 일반적으로 splice()를 사용하는 것이 더 안전하다.

    DOM(Document Object Model)

    DOM은 HTML 문서를 자바스크립트가 이해할 수 있도록 객체 트리 구조로 표현한 것이다. 각 HTML 요소(태그)는 DOM 객체로 변환되어 JS에서 접근·조작이 가능하다.

    // HTML의 <img>, <p>, <button> 같은 요소를 JS에서 객체로 다룰 수 있다.
document.body.style.backgroundColor = "beige";

    DOM 선택자와 접근

    1) 태그 선택

    document.querySelector("p"); // 첫 번째 <p> 요소를 선택

    2) ID 선택자

    document.getElementById("mainTitle");  // ID로 접근
document.querySelector("#mainTitle");  // 동일한 결과 (CSS 선택자 방식)

    3) 클래스 선택자

    document.getElementsByClassName("item"); // 여러 요소 → HTMLCollection(유사 배열)
document.querySelector(".item");         // 첫 번째 요소만 반환

    주의: getElementsByClassName()은 여러 요소를 반환하므로 직접 스타일을 지정하려면 인덱스로 접근해야 한다.

    const boxes = document.getElementsByClassName("box");
boxes[0].style.backgroundColor = "skyblue"; // 첫 번째 요소만 변경

    DOM 스타일 조작

    선택한 요소에 .style.속성명을 지정하면 CSS 속성을 JS로 제어할 수 있다.

    document.querySelector("img").style.border = "2px solid black";
document.getElementById("title").style.color = "red";

    비동기 프로그래밍 개념

    동기(Synchronous)는 작업이 순차적으로 진행되어 하나가 끝나야 다음으로 넘어간다. 비동기(Asynchronous)는 한 작업이 끝나지 않아도 다른 작업이 병행되어 효율이 높다.

    // 예시 (흐름 설명용)
console.log("A 시작");
setTimeout(()=> console.log("B (지연 작업)"), 1000);
console.log("C 종료");

// 출력 순서: A → C → (1초 뒤) B

    자바스크립트는 단일 스레드 언어지만, 비동기 처리를 통해 UI 멈춤 없이 여러 작업을 동시에 다루는 것처럼 동작한다.

    콜백 함수 (Callback)

    콜백 함수는 다른 함수의 인자로 전달되어, 작업이 끝난 후 실행되는 함수다.

    function greet(name, callback){
  console.log("안녕, " + name);
  callback(); // 완료 후 실행
}

function done(){
  console.log("인사 완료!");
}

greet("지민", done); // "안녕, 지민" → "인사 완료!"

    대표적인 비동기 함수인 setTimeout은 콜백을 이용한다.

    setTimeout(() => {
  console.log("3초 후 실행");
}, 3000);

    Promise

    Promise는 비동기 작업의 성공(resolve) 또는 실패(reject)를 표현한다.

    const test = new Promise((resolve, reject) => {
  const isTrue = true;
  if (isTrue) {
    resolve("성공!");
  } else {
    reject("실패!");
  }
});

test
  .then((message) => console.log(message)) // 성공 시
  .catch((error) => console.error(error)); // 실패 시

    Promise 체이닝: .then()은 순차 실행, .catch()는 에러 처리에 사용한다.

    async / await

    async 함수 안에서 await는 Promise가 완료될 때까지 기다린 뒤 다음 코드를 실행한다.

    function delay(ms){
  return new Promise(resolve => setTimeout(resolve, ms));
}

async function run(){
  console.log("시작");
  await delay(2000);  // 2초 기다림
  console.log("2초 후 실행");
}
run();

    즉, await는 Promise의 결과가 반환될 때까지 기다리는 키워드다. 비동기 흐름을 동기식처럼 읽기 쉽게 만든다.

    요약

    • 배열 조작: splice()(추가/삭제/수정), pop()(마지막 삭제), delete(값만 삭제·인덱스 유지).
    • DOM: HTML 문서를 JS 객체 트리로 표현한 것.
    • 요소 선택: getElementById / getElementsByClassName / querySelector.
    • 비동기: callback → Promise → async/await 순으로 발전.
