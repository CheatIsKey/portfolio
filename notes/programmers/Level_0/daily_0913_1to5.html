    문제 2 — 문자열 겹쳐쓰기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    String은 불변 객체이므로, StringBuffer를 사용해서 가변으로 생성 후 replace 메서드를 사용한다. 추가적으로 메서드 체인을 통해 확장이 가능하다.

    문제 3 — 문자열 섞기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    비어있는 StringBuffer를 생성하고 append를 통해 문자열들의 각 char를 추가한다.

    문제 4 — 문자 리스트를 문자열로 변환하기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    반복문을 이용해서 String 리스트 항목을 추가한다. 단, String은 불변 객체이므로 반복할 때마다 새로운 문자열이 힙에 생성되므로 성능에 크게 문제가 된다. 그러므로 StringBuilder나 StringBuffer를 사용하는 것이 좋다.

    문제 5 — 문자열 곱하기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    StringBuffer를 생성하고 반복해서 append로 문자열을 추가한다. 추가로 (String).repeat(count)를 Java11 이후로 지원한다.
