    문제 1 — 더 크게 합치기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    정수를 문자로 변환하고 연결한 뒤, 다시 정수로 변환하여 비교하는 문제다. 추가로 이렇게 풀지 않고 빈 문자열 ""을 넣어서 자바의 특성을 이용한 자동 문자열 전환을 이용해도 됐다. 또한 비교는 Math의 max로 가능했다.

    문제 2 — 두 수의 연산값 비교하기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    이전 문제와 같은 유형으로 정수를 문자로 전환하고 연결한 뒤, 정수로 전환 후 비교해서 출력하는 문제다. 여기서는 빈 문자열을 활용했으며, Math를 활용했다.

    문제 3 — n의 배수

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    "배수"라는 키워드를 확인하고 나머지로 계산했을 때의 결과를 생각하면 된다. 삼항 연산자를 사용하면 더 간결하게 작성이 가능하다.

    문제 링크
