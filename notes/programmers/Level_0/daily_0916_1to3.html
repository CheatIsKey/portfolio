    문제 1 — 원소들의 곱과 합

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    곱셈이 필요한 변수는 1로 초기화하고 덧셈이 필요한 변수는 0으로 초기화 한다. 반복문을 통해 계산된 두 변수를 비교 후, 출력한다.

    문제 2 — 이어 붙인 수

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    StringBuffer를 이용해 홀수, 짝수를 각각 분류해서 코드를 완성했다. 추가로 삼항 연산자로 구현하거나 서로 연결할 때 한 자리씩 밀리는 점을 10의 곱으로 이용하면 StringBuffer 없이 풀 수 있다는 것을 알게 되었다.

    문제 3 — 마지막 두 원소

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    int[]에서는 고정적인 크기를 가지므로, 계산 로직을 따로 구현하고 int[]를 새로 선언한다. 여기서 데이터를 복사할 때, 기존 크기에 한 개를 더 추가한다.
