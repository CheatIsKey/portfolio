    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    2차원 parts 배열을 향상된 for문을 통해서 하나씩 가져온 다음, 각 part 배열에 있는 시작과 끝 인덱스를 할당한다. 그리고 StringBuilder를 통해 문자열을 이어붙인다.

    문제 2 — 문자열의 뒤의 n글자

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    이전 문제와 동일하게 처리하면 된다. my_string의 길이에서 가져오려는 n을 뺀 후, my_string 길이까지 가져오면 된다.

    문제 3 — 접미사 배열

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    my_string의 문자열에서 한 문자씩 늘리면서 자르고 저장하기 위해 List로 선언했다. 또한 Collections의 정렬 기능을 활용했으며, 반환은 Object 타입이 잡히므로 new String[0]를 이용해 문제를 해결한다. 크기를 0으로 줘도 자동적으로 크기에 맞게 생성해준다.
