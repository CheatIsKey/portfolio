    문제 1 — 접미사인지 확인하기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    문자열의 접미사를 전부 잘라내어 리스트에 담고, 그중에서 일치하는 값이 있는지 확인하는 방식으로 문제를 해결했다.

    추가적으로, 불필요한 리스트 생성을 줄이고 substring(i)로 바로 비교하는 방법도 있다. 또한 자바의 내장 메서드인 endsWith()를 사용하면 더 간결하게 접미사 여부를 확인할 수 있다.

    문제 2 — 문자열의 앞의 n글자

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    문자열에서 앞의 n글자를 가져오기 위해 substring(0, n)을 사용했다. 이 방법이 가장 직관적이고 간단하다.

    추가적으로, charAt과 반복문을 이용해 앞에서부터 n개의 문자를 이어 붙이는 방법도 있다. 하지만 이는 다소 비효율적이며, 문자열 인덱스 접근 연습용으로 적합하다.

    문제 3 — 접미사 배열

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    문자열의 앞에서부터 한 글자씩 늘려가며 substring(0, i)로 잘라낸 뒤, is_prefix와 비교하여 접두사 여부를 확인했다. 루프는 i = 1부터 시작해야 올바르게 동작한다.

    추가적으로, 자바에서 제공하는 startsWith() 메서드를 활용하면 보다 간단하게 접두사 여부를 판별할 수 있다.
