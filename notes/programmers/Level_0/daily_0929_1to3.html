    문제 1 — 왼쪽 오른쪽

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    문자열 배열 str_list에서 처음으로 등장하는 "l" 또는 "r"의 위치를 기준으로 배열을 잘라내는 문제다. 규칙은 다음과 같다.

    • "l"을 만나면 그 원소의 왼쪽 부분(자기 자신 제외)을 반환한다.
    • "r"을 만나면 그 원소의 오른쪽 부분(자기 자신 제외)을 반환한다.
    • 배열에 "l"과 "r"이 모두 없으면 빈 배열을 반환한다.

    이를 구현하기 위해 배열을 앞에서부터 순회하며, 조건에 맞는 위치를 찾는 즉시 Arrays.copyOfRange()로 해당 구간을 잘라내 반환한다. 끝까지 순회해도 조건을 만족하지 않으면 빈 배열을 반환한다.

    • Arrays.copyOfRange(str_list, 0, i) : "l"의 왼쪽 부분 반환
    • Arrays.copyOfRange(str_list, i + 1, str_list.length) : "r"의 오른쪽 부분 반환

    시간 복잡도는 배열을 한 번만 순회하므로 O(n), 잘라낸 구간 길이만큼만 새로운 배열이 생성되므로 공간 복잡도도 효율적이다.

    문제 2 — n 번째 원소까지

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    주어진 배열 num_list에서 n번째 원소부터 끝까지 잘라 반환하는 문제다.

    가장 간단한 방법은 자바 표준 라이브러리의 Arrays.copyOfRange()를 활용하는 것이다. 시작 인덱스를 n - 1로 주고 끝 인덱스를 num_list.length로 주면, 원하는 구간을 한 줄로 추출할 수 있다. 

    return Arrays.copyOfRange(num_list, n - 1, num_list.length);

    이렇게 하면 코드가 간결하고 배열 복사도 한 번만 발생한다. 시간 복잡도는 잘라내는 구간의 길이에 비례하는 O(k)이며, 새로운 배열 하나만 생성되므로 공간 복잡도도 O(k)이다.

    참고로, 다음과 같이 직접 for 루프를 돌며 구현할 수도 있다. 이 경우에도 결과는 동일하다. 

    class Solution {
    public int[] solution(int[] num_list, int n) {
        int[] res = new int[num_list.length - (n - 1)];
        for (int i = n - 1, j = 0; i < num_list.length; i++, j++) {
            res[j] = num_list[i];
        }
        return res;
    }
}

    문제 3 — n개 간격의 원소들

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    주어진 배열 num_list에서 n개 간격으로 원소를 뽑아내는 문제다. 예를 들어 n = 3이라면, 인덱스 0, 3, 6, ... 위치의 원소를 차례대로 선택한다.

    구현은 ArrayList에 뽑은 원소들을 담아둔 뒤, 마지막에 stream()과 mapToInt()를 사용해 int[]로 변환하는 방식으로 풀었다.

    시간 복잡도는 뽑히는 원소 개수에 비례하는 O(m)이고, 공간 복잡도는 리스트와 최종 배열에 O(m)이 필요하다.

    추가적으로, 결과 배열의 크기를 미리 계산해 두고 한 번의 for 루프로 바로 채우는 방식도 가능하다. 이 경우 리스트와 스트림 변환 과정이 없어 조금 더 가볍다. 

    class Solution {
    public int[] solution(int[] num_list, int n) {
        int len = num_list.length;
        int m = (len + n - 1) / n; // 뽑히는 원소 개수 = ceil(len / n)

        int[] res = new int[m];
        int idx = 0;
        for (int i = 0; i < len; i += n) {
            res[idx++] = num_list[i];
        }
        return res;
    }
}
