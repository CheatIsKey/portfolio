    문제 1 — 특정 문자열로 끝나는 가장 긴 부분 문자열 찾기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    문자열 myString에서 주어진 패턴 pat이 마지막으로 등장하는 위치까지의 부분 문자열을 반환하는 문제다.

    • String.lastIndexOf(pat) 메서드를 사용해 pat이 myString 내에서 마지막으로 등장하는 인덱스를 찾는다.
    • 찾은 인덱스를 idx라 할 때, 패턴의 마지막 문자까지 포함해야 하므로 myString.substring(0, idx + pat.length())을 반환한다.

    예를 들어, myString = "AbCdEabCdE", pat = "dE"일 경우 pat이 마지막으로 등장하는 위치는 인덱스 7이다. 따라서 substring(0, 7 + 2) → "AbCdEabCdE"를 그대로 반환하게 된다.

    만약 pat이 문자열에 한 번만 등장한다면, 그 지점까지의 부분 문자열만 잘라 반환한다. lastIndexOf()가 패턴을 찾지 못하면 -1을 반환하지만, 문제 조건상 항상 존재한다고 가정할 수 있다.

    문자열을 한 번 검색하고 부분 문자열을 추출하므로 시간 복잡도는 O(n), 공간 복잡도는 O(n)이다.

    핵심은 lastIndexOf()와 substring()을 조합해 “마지막 등장 지점까지 자르기” 로직을 간결하게 구현하는 것이다.

    문제 2 — 문자열이 몇 번 등장하는지 세기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    문자열 myString 안에서 부분 문자열 pat이 겹치더라도 등장할 때마다 모두 세어 그 개수를 반환하는 문제다.

    • indexOf(pat, from) 메서드를 이용해 from 위치부터 pat이 등장하는 인덱스를 찾는다.
    • 패턴이 발견되면(idx != -1), 카운트를 1 증가시키고 from = idx + 1로 갱신해 다음 위치부터 다시 탐색을 이어간다.
    • indexOf()가 -1을 반환하면 더 이상 패턴이 없다는 뜻이므로 반복을 종료한다.

    예를 들어, myString = "aaaa", pat = "aa"인 경우 검색 과정은 다음과 같다: 

    index 0 → "aa" (1회)
      index 1 → "aa" (2회)
      index 2 → "aa" (3회)

    총 3번 등장하므로 3을 반환한다. 즉, 겹치는 구간도 모두 세어주는 것이 특징이다.

    문자열 전체를 순회하며 부분 문자열을 검색하므로 시간 복잡도는 O(n · m) 정도가 되며 (n: myString 길이, m: pat 길이), 별도의 자료구조를 사용하지 않아 공간 복잡도는 O(1)이다.

    핵심은 indexOf()의 두 번째 인자(fromIndex)를 활용해 겹치는 패턴도 탐색할 수 있도록 매번 시작 위치를 1씩 옮겨주는 점이다.

    문제 3 — ad 제거하기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    문자열 배열 strArr에서 원소 중에 "ad"라는 부분 문자열이 포함된 요소를 제거하고 나머지만 남긴 배열을 반환하는 문제다.

    • 빈 리스트 list를 생성한다.
    • 배열의 각 원소 s를 순회하면서 s.contains("ad")로 “ad” 포함 여부를 검사한다.
    • 만약 포함되어 있다면 continue로 건너뛰고, 포함되어 있지 않다면 리스트에 추가한다.
    • 모든 검사를 마친 후 list.toArray(new String[0])로 리스트를 문자열 배열로 변환하여 반환한다.

    예를 들어, strArr = ["apple", "bread", "car", "badge"]인 경우 "bread"와 "badge"는 “ad”를 포함하므로 제외되고, 결과는 ["apple", "car"]가 된다.

    배열을 한 번만 순회하므로 시간 복잡도는 O(n), 새로운 리스트에 일부 문자열만 저장하므로 공간 복잡도는 O(n)이다.

    핵심은 String.contains() 메서드를 이용해 부분 문자열 포함 여부를 간단히 판별하고, 필터링 결과를 새로운 배열로 반환하는 점이다.
