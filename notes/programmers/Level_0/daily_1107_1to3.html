    문제 1 — 컨트롤 제트

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    문자열 s에는 숫자와 "Z"가 공백으로 구분되어 주어진다. 각 숫자는 더해야 하는 값이며, "Z"가 등장할 때마다 바로 직전 숫자를 취소(무효화)하는 규칙이다. 즉, 전체 합계를 계산하되, "Z" 직전에 더했던 값을 다시 빼주는 방식으로 구현한다.

    • 분리 처리: 입력 문자열을 공백 기준으로 나누어 split(" ")으로 순회한다.
    • 취소 처리: 현재 요소가 "Z"면 직전 더했던 값(prev)을 sum에서 빼고 건너뛴다.
    • 누적 합산: 숫자인 경우 정수로 변환해 prev에 저장하고 sum에 더한다.
    • 최종 반환: 모든 토큰을 처리한 뒤 합계 sum을 반환한다.

    예시

    • s = "1 2 Z 3"
      1 → 3 → Z로 2 취소 → +3 ⇒ 결과 4
    • s = "10 20 30 Z Z"
      10+20+30 → Z(30취소) → Z(20취소) ⇒ 결과 10
    • s = "-1 -2 -3 Z"
      -1 -2 -3 → Z(-3 취소) ⇒ 결과 -3

    시간 복잡도

    문자열을 공백 기준으로 한 번 순회하므로 O(n) (n = s.split(" ").length).

    공간 복잡도

    상수 크기의 변수(prev, sum)만 사용하므로 O(1).

    주의

    • 초기값 처리: 첫 숫자 전에는 prev = 0으로 초기화하여 안정적으로 작동.
    • 연속 Z: "Z Z"가 연속될 경우, 두 번째 Z는 이미 취소된 값을 다시 취소하지 않도록 prev가 0이 되므로 영향 없음.
    • 숫자 변환: 문자열을 정수로 변환할 때 Integer.parseInt(str) 사용.
    • 예외 입력: 비어 있거나 숫자 외 문자열은 문제 조건상 등장하지 않음.
    대안 아이디어
    • 스택 활용 방식 — “되돌리기” 개념을 명시적으로 처리
      
      import java.util.*;
      
      class Solution {
          public int solution(String s) {
              Stack<Integer> stack = new Stack<>();
              for (String str : s.split(" ")) {
                  if (str.equals("Z")) {
                      if (!stack.isEmpty()) stack.pop();
                  } else {
                      stack.push(Integer.parseInt(str));
                  }
              }
              return stack.stream().mapToInt(Integer::intValue).sum();
          }
      }

      스택에 숫자를 쌓고, "Z"가 나오면 pop()으로 직전 항목 제거.

    • reduce 기반 스트림 처리 — 함수형 접근
      
      import java.util.*;
      import java.util.stream.*;
      
      class Solution {
          public int solution(String s) {
              Deque<Integer> dq = new ArrayDeque<>();
              for (String str : s.split(" ")) {
                  if (str.equals("Z")) {
                      if (!dq.isEmpty()) dq.removeLast();
                  } else {
                      dq.addLast(Integer.parseInt(str));
                  }
              }
              return dq.stream().mapToInt(Integer::intValue).sum();
          }
      }

      스택 대신 Deque를 사용해 후입선출 구조를 유지하면서도 스트림 합산으로 마무리한다.

    문제 2 — 소인수분해

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    자연수 n의 소인수를 모두 구하되, 중복 없이 오름차순으로 반환하는 문제다. 2부터 시작해 √n까지 나누며, 나누어 떨어질 때마다 n을 나누고 해당 인수를 Set에 저장한다. 반복 후 n이 1보다 크면 마지막 남은 수 또한 소수이므로 추가한다.

    • 소인수 분해 반복: 2부터 i * i ≤ n까지 순회하며 n % i == 0이면 나누기 반복.
    • 중복 제거: LinkedHashSet 사용으로 중복 방지 및 입력 순서(오름차순) 유지.
    • 마지막 처리: 루프 후 n > 1이면 남은 값은 소수 → 추가.
    • 출력 변환: Set을 스트림으로 변환해 int[] 배열로 반환.

    예시

    • n = 12
      12 → 2로 나눔 → 6 → 2로 나눔 → 3 → 3으로 나눔 → 1
      소인수: [2, 3]
    • n = 420
      2, 3, 5, 7 순으로 나누어짐 → [2, 3, 5, 7]
    • n = 17
      17은 소수 → [17]

    시간 복잡도

    소인수 탐색은 최대 √n까지 수행되므로 O(√n).

    공간 복잡도

    소인수 개수에 비례해 O(k) (k는 서로 다른 소인수의 개수). 일반적으로 O(log n) 이하.

    주의

    • 중복 방지: 동일 소인수를 여러 번 나누더라도 Set에 한 번만 저장된다.
    • LinkedHashSet 선택: 삽입 순서를 유지하므로 결과 배열이 자동으로 오름차순 정렬.
    • 마지막 소수 처리: 반복 후 n > 1이면 그 자체가 소수임을 잊지 말 것.
    • 예외 입력: n < 2인 경우는 문제 조건상 등장하지 않음.
    대안 아이디어
    • 정렬된 리스트 사용 — 단순화된 구조
      
      import java.util.*;
      
      class Solution {
          public int[] solution(int n) {
              List<Integer> list = new ArrayList<>();
              for (int i = 2; i * i <= n; i++) {
                  if (n % i == 0) {
                      list.add(i);
                      while (n % i == 0) n /= i;
                  }
              }
              if (n > 1) list.add(n);
              return list.stream().mapToInt(Integer::intValue).toArray();
          }
      }

      중복 제거를 직접 조건문으로 수행해 List만으로 해결 가능.

    • 에라토스테네스 기반 접근 — 사전 계산형
      
      import java.util.*;
      
      class Solution {
          public int[] solution(int n) {
              boolean[] prime = new boolean[n + 1];
              Arrays.fill(prime, true);
              prime[0] = prime[1] = false;
              for (int i = 2; i * i <= n; i++) {
                  if (prime[i]) {
                      for (int j = i * i; j <= n; j += i)
                          prime[j] = false;
                  }
              }
      
              List<Integer> result = new ArrayList<>();
              for (int i = 2; i <= n; i++) {
                  if (prime[i] && n % i == 0)
                      result.add(i);
              }
              return result.stream().mapToInt(Integer::intValue).toArray();
          }
      }

      미리 소수를 구해두고 n을 나누는 소수만 추출하는 방식.

    문제 3 — 숨어있는 숫자의 덧셈 (1)

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    문자열 my_string에 포함된 모든 숫자 문자(‘0’~‘9’)를 찾아 그 합을 구하는 문제다. 문자를 하나씩 순회하며 숫자인 경우 정수값으로 변환해 누적 합계를 계산한다.

    • 문자 순회: toCharArray()로 문자열을 문자 배열로 변환하고 반복 처리.
    • 숫자 판별: 문자 c가 '0' <= c <= '9' 범위에 있으면 숫자로 간주.
    • 정수 변환: 문자 ‘0’의 코드 값을 빼서 실제 숫자로 변환 → c - '0'.
    • 합산: 변환된 값을 sum에 더함.
    • 결과 반환: 전체 문자열 순회를 마친 뒤 최종 합계 sum을 반환.

    예시

    • my_string = "aAb1B2cC34oOp"
      숫자: 1, 2, 3, 4 → 합계 10
    • my_string = "1a2b3c4d123"
      숫자: 1, 2, 3, 4, 1, 2, 3 → 합계 16
    • my_string = "abc"
      숫자 없음 → 합계 0

    시간 복잡도

    문자열 길이를 n이라 할 때, 한 번 순회하므로 O(n).

    공간 복잡도

    상수 크기의 변수(sum)만 사용하므로 O(1).

    주의

    • 연속된 숫자 처리: 각 자리 숫자를 독립적으로 더하므로 “12”는 1 + 2로 계산됨.
    • 비숫자 문자: 알파벳, 특수문자 등은 무시된다.
    • 음수 부호: ‘-’는 숫자가 아니므로 포함되지 않는다.
    • 공백 처리: 공백 역시 무시된다.
    대안 아이디어
    • 정규식 이용 — 숫자 추출 단축 표현
      
      import java.util.*;
      
      class Solution {
          public int solution(String my_string) {
              return my_string.replaceAll("[^0-9]", "")
                              .chars()
                              .map(c -> c - '0')
                              .sum();
          }
      }

      정규식 [^0-9]로 숫자가 아닌 문자를 제거하고 남은 숫자들을 합산.

    • 스트림 기반 처리 — 가독성 중심 함수형 접근
      
      import java.util.*;
      
      class Solution {
          public int solution(String my_string) {
              return (int) my_string.chars()
                                    .filter(Character::isDigit)
                                    .map(c -> c - '0')
                                    .sum();
          }
      }

      문자 스트림에서 Character.isDigit으로 필터링하고 각 문자를 숫자로 변환해 합산한다.
