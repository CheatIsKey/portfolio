    [LV0] 문자 출력

    문제

    문자열 str이 주어지면 그대로 출력하세요.

    정답 코드 (Java)

    import java.io.*;
class Solution1 {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String str = br.readLine();
        System.out.print(str);
    }
}

    해설

    입출력 기초 문제. 버퍼를 쓰면 대용량 입력에서도 안정적입니다.

    [LV0] 문자열 반복

    문제

    문자열 my_string을 n번 반복해 이어붙인 문자열을 반환하세요.

    정답 코드 (Java)

    class RepeatString {
    public String solution(String my_string, int n) {
        StringBuilder sb = new StringBuilder(my_string.length() * n);
        for (int i = 0; i < n; i++) sb.append(my_string);
        return sb.toString();
    }
}

    해설

    StringBuilder로 누적 연결하면 불필요한 임시 문자열 생성을 줄일 수 있습니다.

    [LV0] 배열 두 배 만들기

    문제

    정수 배열 numbers의 모든 원소에 2를 곱한 배열을 반환하세요.

    정답 코드 (Java)

    import java.util.*;
class DoubleArray {
    public int[] solution(int[] numbers) {
        int[] ans = new int[numbers.length];
        for (int i = 0; i < numbers.length; i++) ans[i] = numbers[i] * 2;
        return ans;
    }
}

    해설

    단순 순회. 스트림을 써도 되지만 반복문이 가장 가볍고 빠릅니다.

    [LV0] 짝수 홀수 개수

    문제

    정수 배열에서 짝수 개수와 홀수 개수를 배열로 반환하세요. (순서: [짝수, 홀수])

    정답 코드 (Java)

    class EvenOddCount {
    public int[] solution(int[] num_list) {
        int even = 0, odd = 0;
        for (int n : num_list) {
            if ((n & 1) == 0) even++;
            else odd++;
        }
        return new int[]{even, odd};
    }
}

    해설

    비트 연산 & 1로 홀짝 판정하면 분기 비용이 거의 없습니다.

    [LV0] 피자 나눠 먹기 (1)

    문제

    피자 한 판에 7조각. 사람 수 n이 주어지면, 모두가 한 조각 이상 먹기 위한 최소 피자 수를 구하세요.

    정답 코드 (Java)

    class Pizza {
    public int solution(int n) {
        return (n + 6) / 7; // 올림: ⌈n / 7⌉
    }
}

    해설

    올림 나눗셈 패턴 (a + b - 1) / b를 활용합니다. 여기서는 b = 7.
