    문제 1 — 정수 제곱근 판별

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    정수 제곱근 판별 (프로그래머스 12934)는
    어떤 양의 정수 n이 정수의 제곱인지 확인하고, 맞다면 그 다음 정수의 제곱을, 아니라면 -1을 반환하는 문제다.

    • n이 어떤 정수 r에 대해 r²이라면 → (r+1)² 반환
    • 그렇지 않다면 → -1 반환

    전략 개요

    • 1. 제곱근 구하기
      • Math.sqrt(n)을 이용해 n의 제곱근을 구한다.
      • double 결과를 long으로 캐스팅해서 정수 부분만 떼온다. 
        long r = (long) Math.sqrt(n);
    • 2. 정수 제곱인지 확인
      • r * r이 원래 값 n과 정확히 같은지 비교한다.
      • 같다면 n은 어떤 정수의 완전제곱이다. 
        if (r * r == n) {
          // n은 r² 형태
      }
      • 다르면 정수의 제곱이 아니므로 -1을 반환한다.
    • 3. 결과 반환
      • 정수 제곱이라면, 문제 조건에 따라 (r+1)²를 반환한다. 
        if (r * r == n) {
          return (long) Math.pow(r + 1, 2);
      } else {
          return -1;
      }

    코드가 하는 일 요약

    • long r = (long) Math.sqrt(n);
      • n의 제곱근을 구해서 정수 부분만 r에 담는다.
    • if (r * r == n)
      • r²가 정확히 n과 같다면, n은 완전제곱수.
      • 이때 (r+1)²를 반환 → 다음 완전제곱수.
    • 그 외의 경우
      • r² ≠ n이면 정수 제곱이 아니므로 -1 반환.

    시간 · 공간 복잡도

    • 시간 복잡도
      • 연산은 제곱근 계산 및 상수번의 산술·분기 뿐 → O(1).
    • 공간 복잡도
      • 추가 변수만 사용하므로 O(1).

    문제 2 — 정수 내림차순으로 배치하기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    정수 내림차순으로 배치하기 (프로그래머스 12933)는
    주어진 정수 n의 각 자릿수를 내림차순으로 정렬한 뒤, 다시 하나의 수로 만들어 반환하는 문제다.

    • n을 10으로 나누면서 각 자릿수를 분리한다.
    • 자릿수들을 내림차순으로 정렬한다.
    • 정렬된 자릿수를 앞에서부터 이어붙여 새로운 수를 만든다.

    전략 개요

    • 자릿수 분리
      • n % 10으로 마지막 자릿수를 얻고, n /= 10으로 한 자리 줄인다.
      • 이 과정을 반복하며 모든 자릿수를 리스트에 저장한다.
    • 내림차순 정렬
      • 자릿수 리스트를 Collections.reverseOrder() 기준으로 내림차순 정렬한다.
    • 수 재구성
      • result = result * 10 + digit 방식으로 자릿수를 순서대로 붙인다.

    코드가 하는 일

    • 자릿수 추출 
      List<Integer> digits = new ArrayList<>();
      
      while (n > 0) {
          digits.add((int)(n % 10));
          n /= 10;
      }
      • n의 각 자릿수를 뒤에서부터 하나씩 뽑아 digits에 저장한다.
    • 내림차순 정렬 
      digits.sort(Collections.reverseOrder());
      • 자릿수들을 큰 값부터 오도록 정렬한다.
    • 정렬된 자릿수로 결과 만들기 
      long result = 0;
      for (int i : digits) {
          result = result * 10 + i;
      }
      
      return result;
      • 앞에서부터 자릿수를 붙여 최종 숫자를 완성한다.
      • long 범위에서 안전하게 결과를 반환한다.

    시간 · 공간 복잡도

    • 시간 복잡도
      • 자릿수 개수를 d라 하면, 추출은 O(d), 정렬은 O(d log d), 재구성은 O(d).
      • 따라서 전체는 O(d log d).
    • 공간 복잡도
      • 자릿수 리스트를 저장하므로 O(d).
