    문제 1 — 자연수 뒤집어 배열로 만들기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    자연수 뒤집어 배열로 만들기 (프로그래머스 12932)는
    주어진 자연수 n의 각 자리 숫자를 뒤에서부터(일의 자리부터) 뽑아 배열에 담아 반환하는 문제다.

    • 예: 12345 → [5, 4, 3, 2, 1]
    • 핵심은 n % 10으로 마지막 자리를 얻고, n /= 10으로 한 자리씩 줄이는 것.

    전략 개요

    • 1) 배열 크기 구하기
      • 정답 배열은 자릿수만큼 필요하므로, 임시 변수 tmp로 자릿수를 센다.
      • tmp > 0 동안 tmp /= 10을 반복하면 자릿수만큼 카운트 증가.
    • 2) 뒤집어서 채우기
      • n에서 n % 10을 꺼내 배열에 넣고
      • n /= 10으로 자릿수를 하나 줄인다.
      • 이 과정을 n이 0이 될 때까지 반복하면 자연스럽게 “뒤집힌 순서”로 들어간다.

    코드가 하는 일

    • 자릿수 계산 
      int length = 0;
      long tmp = n;
      
      while (tmp > 0) {
          length++;
          tmp /= 10;
      }
      • tmp를 10으로 계속 나누면서 자릿수를 센다.
      • 이 값이 곧 정답 배열의 길이.
    • 배열 채우기 (일의 자리부터) 
      int idx = 0;
      int[] answer = new int[length];
      
      while (n > 0) {
          answer[idx++] = (int) (n % 10);
          n /= 10;
      }
      • n % 10 → 현재 일의 자리
      • 배열에 넣고 n /= 10으로 다음 자리로 이동
      • 결과적으로 뒤집힌 순서로 저장된다.

    시간 · 공간 복잡도

    • 시간 복잡도
      • 자릿수 계산 O(d) + 배열 채우기 O(d) → O(d)
      • 여기서 d는 자릿수(최대 19자리 수준)
    • 공간 복잡도
      • 정답 배열 크기만 사용 → O(d)

    문제 2 — 자릿수 더하기

    문제 링크

    프로그래머스 문제 보기

    코드 (IDE 스크린샷)

    해설

    자릿수 더하기 (프로그래머스 12931)는
    정수 n이 주어졌을 때, 각 자리 숫자의 합을 구하는 문제다.

    • 핵심은 일의 자리부터 하나씩 분리해서 누적하는 것.
    • n % 10으로 마지막 자릿수를 얻고, n /= 10으로 자릿수를 하나 줄인다.

    전략 개요

    • 반복 처리
      • n이 0이 될 때까지 반복한다.
      • 매 반복마다
        1. n % 10 → 일의 자리 추출 후 합에 더하기
        2. n /= 10 → 다음 자리로 이동

    코드가 하는 일

    • 자릿수 누적 
      int sum = 0;
      
      while (n > 0) {
          sum += n % 10;
          n /= 10;
      }
      • n % 10으로 현재 일의 자리를 얻어 sum에 누적한다.
      • n /= 10으로 맨 뒤 자릿수를 제거하며 다음 반복에서 그 다음 자리를 처리한다.
      • 반복이 끝나면 sum이 곧 모든 자릿수의 합이다.

    시간 · 공간 복잡도

    • 시간 복잡도
      • n의 자릿수를 d라고 하면, 한 자리씩 처리하므로 O(d).
    • 공간 복잡도
      • 추가 배열 없이 변수만 사용하므로 O(1).
